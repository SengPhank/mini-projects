## About
> Basic Minesweeper game that will later be interacted with a a Constraint Satisfaction Problem solver.

---

## Requirements

- [C++](https://en.cppreference.com/w/cpp/17)
- [g++ 11.4.0](https://gcc.gnu.org/) or equivalent
- [raylib](https://www.raylib.com/)
- GNU Make

## Build and running

```bash
make build // Build
make run   // Run
```
